#### bool CreateFile(nType);

Creates a new file. The type of the file which will be created needs to be set.

## Parameters:

| Name         | Type             | Description                                                                                                                                                                         |
| ------------ | ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| *sExtension* | constÂ wchar\_t\* | The file extension. The following values are possible: **docx**, **xlsx** or **pptx** (see [AVS\_OFFICESTUDIO\_FILE\_XXX](/docbuilder/integrationapi/default#format-types) values). |

## Example

#### C++

```
std::wstring sWorkDirectory = NSUtils::GetBuilderDirectory();
CDocBuilder::Initialize(sWorkDirectory.c_str());
CDocBuilder oBuilder;
oBuilder.CreateFile("docx");
CDocBuilder::Dispose();
```

#### .docbuilder

```
builder.CreateFile("docx");
```
